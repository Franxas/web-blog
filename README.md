This is my website project. I made it mostly to share my work and centralize my content from across all diferent platforms.    
I developed it in Node Javascript as a single page application and tried to implement an MVC arquitectural aproach as much as possible. I also made use of the JS E6 new features so I could treat this purely as an OOP pproject.  
The home and portfolio pages is static.  
There is a Post page that works as my personal blog hosted on my website. I can add, update and delete entries by loging in from an admin url and I'm using MongoDB to store the data and manage from the website.  
There is a synth one can play around with. This is a patch I made in Max rnbo and exported it for the web so I can ghave access to it through the RNBO Api.
